# Domain Modules

Thư mục này giữ kiến trúc module độc lập với framework để có thể tái sử dụng ở backend, frontend hoặc worker sau này.

| Module | Trách nhiệm |
| --- | --- |
| genealogy | Cây gia phả, quan hệ huyết thống, đời, nhánh |
| i18n | Đa ngôn ngữ, IPA, Pinyin, bản dịch nội dung |
| lunar | Lịch âm, ngày giỗ, ngày lễ truyền thống |
| canchi | Thiên Can, Địa Chi, con giáp |
| zodiac | 12 con giáp và cung hoàng đạo phương Tây |
| numerology | Báo cáo thần số học tham khảo |
| bazi | Bát tự tham khảo văn hóa |
| iching | Kinh Dịch và quẻ tham khảo |
| astrology | Chiêm tinh phương Tây tham khảo |
| media | Ảnh, video, tài liệu, album |
| events | Sinh nhật, ngày giỗ, họp mặt, lịch gia đình |
| auth | Xác thực, phân quyền, chính sách truy cập |
| export | Xuất JSON, CSV, PDF, GEDCOM tương lai |
| backup | Sao lưu, phục hồi, kiểm tra toàn vẹn dữ liệu |
